# Spatial Transcriptomics and Cancer Thesis
This is for a thesis project with Professor Matina Donaldson-Matasci for Harvey Mudd College and Carsten Daub at the Karolinska Institute.
The repository is at [here](https://github.com/beelabhmc/spatial_cancer_thesis) this is just a copy. This thesis sought to translate images of H&E stained cancer tumor to spot specific pathway expressions.
